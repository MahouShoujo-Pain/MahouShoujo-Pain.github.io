云计算

很容易理解什么是计算，但是云计算就比较难理解了。

用喝水（获取水资源）举例子，以前想要喝水需要自己打井，想要获取计算资源需要自己组装电脑，属于裸机。

但是现在喝水不需要自己打井了，想要喝水打开水龙头就可以获取水资源，想要获取计算资源也仅仅在网页上连接远程服务就可以了。

而其中云计算有分为Iaas，paas，saas。

其中紫色是用户可以控制的部分，从Iaas到Saas，用户可自由控制的资源越来越少，但用户也越来越不用关心资源。只关注自身需要使用的部分。

磁盘
  • 协议：SCSI、SATA、SAS、SSD
磁盘阵列

  • RAID是英文Redundant Array of Independent Disks的缩写，中文简称为独立冗余磁盘阵列。简单的说，RAID是一种把多块独立的硬盘（物理硬盘）按不同的方式组合起来形成一个硬盘组（逻辑硬盘），从而提供比单个硬盘更高的存储性能和提供数据备份技术。

  • raid0 分割存储

  • raid1 备份存储

  • raid5 分块存储+奇偶校验

系统
  • 系统 = 内核 + （解释器）shell + 应用程序

分区
  • 标准分区

  1. 系统盘：/ boot 引导分区 200M

  2. 交换区：swap 交换分区 内存不够、硬盘来凑 建议大小1.5*内存(<8G)

  3. 根分区：/ 剩余空间

  • 数据分区

    1. / boot —–200M

    2. swap —–<8G

    3. / —–20G

    4. /data —–剩余空间

  • 灵活分区

    1. / boot —–200M

    2. swap —–<8G

    3. / —–20~200G

    4. 先不管 —–剩余空间
